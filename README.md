#  Real-Time Gesture Drawing Board

##  Overview
Real-Time Gesture Drawing Board is a computer vision-based application that allows users to draw on a virtual canvas using hand gestures. It uses real-time hand tracking to replace traditional input devices like a mouse or stylus.

---

##  Features
-  Draw using hand gestures  
-  Color selection with gestures  
-  Erasing functionality  
-  Undo / Redo support  
-  Canvas control (clear, reset, etc.)  
-  Real-time performance  

---

##  Tech Stack
- Python  
- OpenCV  
- MediaPipe  
- NumPy  

---


---

##  How It Works
- Uses MediaPipe for hand tracking  
- Detects finger positions in real-time  
- Maps gestures to drawing actions  
- Renders drawing on a virtual canvas using OpenCV
